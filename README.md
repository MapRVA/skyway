# skyway

skyway is a command-line OpenStreetMap data conversion and filtering utility.

## Documentation

Please see the [skyway documentation](https://maprva.github.io/skyway/) to learn more!

## Supported Formats

| Format        | Reader | Writer |
| --------------|--------|--------|
| GeoJSON       | ➖     | ❌     |
| o5m           | ❌     | ❌     |
| OPL           | ✅     | ✅     |
| OSM JSON      | ✅     | ✅     |
| Overpass JSON | ✅     | ❌     |
| OSM XML       | ✅     | ✅     |
| PBF           | ✅     | ❌     |

## Contributing
